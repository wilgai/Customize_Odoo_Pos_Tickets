Remove Decimal Zero Trailing

Overview

This module enable you to remove decimal zero trailing in all Odoo views and reports.

Features
Remove decimal zero trailing for float and monetary fields in all Odoo Views.
Remove decimal zero trailing for float and monetary fields in all Odoo HTML and PDF reports.
How it works

You need only to install the module without any further configuration.

  • If the amount is a net ten thousand, then it will be look like this 10,000 instead of this 10,000.000
  • If the amount is ten thousand and five cents, then it will be look like this 10,000.05 instead of this 10,000.050
  • If the amount is ten thousand and fifteen cents, then it will be look like this 10,000.15 instead of this 10,000.150
  • If the amount is ten thousand and one hundred fifteen cents, then it will be look like this 10,000.115
